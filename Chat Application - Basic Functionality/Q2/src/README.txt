Python version : 2.7.12

Instructions on how to run :

1) Ensure that all .py files are within the same directory

2) In order to start the server, execute the following command :

python server.py --sp  9090

3) To start up a client, use : 

python client.py --sip localhost --sp 9090

The port number of the above two commands must match

4) Any number of clients could be started by invoking the above command
